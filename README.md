# WebOmok
[서버 프로젝트](https://github.com/floweryclover/web-omok-server)

* 고도엔진4로 작성된 오목게임
* 서버에 다수의 클라이언트가 접속하여 각자 방 생성후 플레이하는 방식
* GDScript로 작성, 웹소켓을 이용하여 웹에서 플레이 가능
